职业总目标（3 年）意向城市：深圳广州

从“AI Agent 实习生”成长为“能独立负责 AI 应用模块的技术骨干”
短期（0–1 年）：站稳 AI 应用开发岗位，形成稳定交付能力
中期（1–2 年）：在 Prompt / RAG / Agent 框架中深耕一个方向，具备独立设计与优化能力
长期（2-3 年）：成为 AI 应用技术负责人 / 技术型产品经理 / 独立 AI 创作者

三、分阶段规划
第一阶段：实习期（0–6 个月）
关键词：上手、交付、建立信任
技术目标
掌握至少一种主流大模型 API（OpenAI / Claude / 国内大模型）
熟练使用 Prompt Engineering（少样本、思维链、角色设定）
了解 RAG 基础：向量数据库、Embedding、检索流程
掌握一门 Agent 框架：LangChain / LlamaIndex（任选其一）
后端基础：Python + FastAPI 或 Node.js + Express
差异化动作（发挥优势）
用写作能力沉淀：Prompt 模板库、接口文档、使用说明
用设计和共情力优化：Agent 交互流程、错误提示、用户体验

阶段产出
参与至少 1 个 AI Agent 项目（内部工具 / 对外 Demo）
GitHub 上有 1–2 个 AI 相关项目（含 README 说明）
第二阶段：入职第 1 年（6–18 个月）
关键词：独立负责、深耕细分
方向选择（结合公司业务三选一）
Prompt & Agent 策略：行为设计、工具调用链路优化
RAG & 知识库：企业知识问答、长文本处理

AI 应用全栈：前端 + 后端 + 模型对接

能力提升
能独立负责一个小型 AI 功能模块
能从“实现功能”进阶到“评估效果”（准确率、成本、延迟）
能用写作能力做内部分享、技术博客

第三阶段：第 1-2 年（高级工程师）
关键词：系统思维、技术决策
理解 LLM 基础原理（Transformer、微调、推理优化概念）
能对比不同模型/方案的性价比
参与技术选型、Code Review、新人指导
持续输出：开源项目、技术文章、行业交流

AI 应用技术负责人
